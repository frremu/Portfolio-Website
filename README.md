# Umer Farooq - 3D Developer Portfolio

Welcome to my interactive 3D Portfolio built with React, Three.js, React Three Fiber, GSAP, and Vite!

## Features
- **3D Abstract Interactive Setup:** Built precisely using `@react-three/fiber` and `@react-three/drei`.
- **GSAP Animations:** Dynamic text revealing and smooth scroll logic using `ScrollTrigger`.
- **Custom Projects Carousel:** Beautifully highlighting my professional enterprise projects like MedCo, Shifa, and OneClicks. 
- **Dynamic Mouse Tracking System:** Parallax hover interactions mapped completely across the client view.

## Run Locally

```bash
npm install
npm run dev
```
